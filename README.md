# React shop

<h1 align="center">Frontend</h1>
Technologies: React.js, Bootstrap <br>

**Run front:** 

```
npm start
```

**Address:**

```
localhost:3000/
```

<h1 align="center">Backend</h1>
Technologies: Node.js, Express

**Run back:**

```
npm run dev
```

**Address:**

```
localhost:8080/
```

<h1 align="center">Databases</h1>
Technologies: Postgresql, sequelize

**Address:**

```
localhost:5432/
```
